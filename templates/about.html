Hướng dẫn sử dụng

  1. Upload ảnh tế bào máu: Chọn file ảnh từ máy tính của bạn
  2. Upload Ground Truth: Nếu có file label YOLO format (.txt), bạn có thể upload để so sánh kết quả
  3. Nhấn nút Phân tích: Hệ thống sẽ tự động xử lý và trả về kết quả
  4. Xem kết quả:
    • Số lượng và tỷ lệ từng loại tế bào
    • Hình ảnh với bounding boxes
    • So sánh với ground truth (nếu có)
    • Các metrics đánh giá (Precision, Recall, F1)

Format file Ground Truth

Định dạng YOLO Label

File label cần tuân theo định dạng YOLO với mỗi dòng chứa thông tin một bounding box:

class_id center_x center_y width height

Trong đó:

  • class_id: ID của class theo format
    • 0: TJ
    • 1: TA
    • 2: S
    • 3: G
    • 4: Healthy
    • 5: Others
    • 6: Diff
  • center_x, center_y: Tọa độ tâm của box (chuẩn hóa 0-1)
  • width, height: Kích thước của box (chuẩn hóa 0-1)
Ví dụ:
4 0.512 0.487 0.154 0.132    # Healthy cell
0 0.234 0.765 0.098 0.112    # TJ cell
3 0.678 0.345 0.123 0.145    # G cell
